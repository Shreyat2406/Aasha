# AASHA Cyber Safety App

## Description
AASHA is an AI-powered cyber safety assistant. Users can:
- Chat with AASHA about online safety, scams, and phishing.
- Complete learning modules and quizzes.
- Scan URLs and files for cyber threats.
- Track points, levels, and stats.

Built with Next.js 13+, Firebase, TailwindCSS, and shadcn/ui.

---

## Technologies
- Next.js 13+ (App Router)
- TypeScript
- Firebase (Auth + Firestore)
- TailwindCSS + shadcn/ui
- Lucide Icons
- Vercel Deployment

---

## Features
- **AI Chat:** Ask AASHA questions about cyber security.
- **Learning Modules:** Interactive content with quizzes.
- **Scan Feature:** Scan URLs or files to detect potential cyber threats.
- **Points & Levels:** Users earn points for completing modules, quizzes, and scans.
- **User Profile:** Track stats, modules completed, quizzes, chat history, and scans.
- **Theme Toggle:** Light/Dark mode.
